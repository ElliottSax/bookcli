**Six Fundamentals:**
1. Pattern sensitivity - feeling threads without visual confirmation
2. Thread discrimination - distinguishing types by sensation
3. Corruption assessment - active vs dormant, malignant vs benign
4. Energy regulation - sustained channeling without damage
5. Surgical precision - minimum intervention cuts
6. Emergency protocols - when to heal, when to cut, when to run

**Thread Sensations:**
- Life threads: warm
- Memory threads: cool, flowing
- Emotional connections: pulsing
- Physical bonds: solid, static

**Minimum Intervention Philosophy:**
- Remove only corrupting elements
- Preserve healthy connections
- Both patients survive when possible
- Relationship survives minus corruption
